# KeretaKitaApp

Aplikasi Pemesanan Travel Kereta Kita merupakan aplikasi mobile yang melayani pemesanan travel ke pengguna. 
Fitur-fitur yang ada pada Kereta Kita App :
1. Profile Pengguna
2. Booking Pemesanan
3. History
4. About Us
5. Penggunaan SharedPreferences dan CRUD SQLite
